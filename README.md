**Install**

```
sudo ./install.sh
```

**Features**

- Java - ```CocInstall coc-java```
- Python - ```CocInstall coc-python```
- JSON - ```CocInstall coc-json```
- Typescript - ```CocInstall coc-tsserver```
- Snippets - ```CocInstall coc-snippets```
