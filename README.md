# dotfiles

chezmoi source directory. Debian and macOS, zsh, Claude Code, spoken replies.

**Bootstrapping a new machine?** Read the guide first, it blocks on human input four
times and installs more than dotfiles:

```sh
curl -s https://raw.githubusercontent.com/w1am/dotfiles/main/INSTALLATION.md
```

**Editing what's already installed?** Read `CLAUDE.md` at the repo root. It covers the
filename prefixes, the source-not-target rule, and the secrets policy.
