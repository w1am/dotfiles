# Dotfiles

Personal dotfiles managed with [chezmoi](https://www.chezmoi.io/). This repo is the
chezmoi **source directory** (`~/.local/share/chezmoi`). `README.md` and `CLAUDE.md`
are symlinks to this file — edit `AGENTS.md` only.

## Quick start (new machine)

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply w1am
```

This installs chezmoi, clones this repo, and applies everything: installs packages,
clones Oh My Zsh + plugins, installs Neovim, sets zsh as the default shell, and
generates a per-machine SSH key. The only manual step is pasting the printed SSH
public key into GitHub (or running the `gh ssh-key add` command it prints).

## Layout

| Source path | Target / effect |
|-------------|-----------------|
| `dot_zshrc` | `~/.zshrc` — Oh My Zsh (agnoster theme, spelling correction, history, plugins); prepends `~/.local/bin` to `PATH` |
| `dot_tmux.conf` | `~/.tmux.conf` — backtick prefix, vi mode, Wayland clipboard via `wl-copy` |
| `private_dot_config/nvim/` | `~/.config/nvim` — lazy.nvim-based Neovim config |
| `.chezmoiexternal.toml` | Clones Oh My Zsh + `zsh-autosuggestions`, `zsh-syntax-highlighting`, `zsh-completions` |
| `.chezmoiscripts/` | Bootstrap scripts (run during `chezmoi apply`) — see below |
| `.chezmoiignore` | Keeps `AGENTS.md`/`README.md`/`CLAUDE.md` out of `$HOME` |

### Bootstrap scripts (`.chezmoiscripts/`)

Scripts are split by **lifecycle**, and chezmoi runs them in numeric order:

| Script | Type | Purpose |
|--------|------|---------|
| `run_onchange_after_10-install-packages.sh` | `run_onchange_` | apt-installs `zsh git gh tmux wl-clipboard`. Re-runs whenever the package list changes. Detects via `dpkg` (command name may differ from package name, e.g. `wl-clipboard` → `wl-copy`). |
| `run_onchange_after_15-install-neovim.sh` | `run_onchange_` | Installs the latest stable Neovim from the official release tarball into `~/.local` (no sudo). Bump `NVIM_CHANNEL` to update. |
| `run_once_after_20-set-default-shell.sh` | `run_once_` | `chsh` to make zsh the default shell. |
| `run_once_after_30-generate-ssh-key.sh` | `run_once_` | Generates a per-machine ed25519 SSH key (never synced) and prints it for GitHub. |

`run_once_` = one-time per machine. `run_onchange_` = re-runs when the script's
contents change. All scripts are idempotent and safe to re-run.

## Conventions

- **Edit the source, not the target.** Use `chezmoi edit ~/.zshrc`, then `chezmoi apply`.
- **Capture a change made directly to a dotfile** with `chezmoi add ~/.zshrc`.
- **SSH keys are never committed** — they are per-machine identity, generated by the
  bootstrap script. Do not add private keys to this repo (even encrypted, unless
  intentionally using chezmoi's secret support).
- **Neovim is installed from the upstream tarball, not apt**, to track the latest
  release. apt's version can lag on older distros.
- **Add a package** by editing the `packages=` list in the install-packages script;
  `run_onchange_` makes it re-run on every machine.

## Common commands

```sh
chezmoi edit ~/.zshrc        # edit a managed file
chezmoi diff                 # preview pending changes
chezmoi apply                # apply changes to the live system
chezmoi add ~/.somefile      # start managing an existing file
chezmoi cd                   # open a shell in the source dir (this repo)
```

## Environment notes

- Target OS: Ubuntu (apt-based). Scripts no-op gracefully on non-Debian systems.
- `sudo` is not passwordless here — apt and `chsh` will prompt for a password.
